I'm the Web Developer

Hi! Nama saya Brandon Alexandro Sutanto, seorang siswa kelas 11H dengan nomor absen 05. Saya lahir pada 22 September 2008, dan saat ini berusia 16 tahun. Saya berasal dari Malang dan memiliki ketertarikan besar di bidang matematika, fisika, kimia, dan informatika. Saat ini, saya sedang aktif mengerjakan sebuah proyek web bernama Brandon Manga, sebuah platform yang berfokus pada penjualan dan distribusi manga ke seluruh dunia.

Sebagai seorang murid informatika kelas 2INF04, tentunya saya bertanggung jawab untuk merancang, membangun, dan mengelola website Brandon Manga agar dapat memberikan pengalaman terbaik bagi pengguna. Dengan kombinasi antara kehebatan dan ketampanan saya sebagai murid dan pembaca manga, saya berharap dapat memberikan kontribusi positif dalam dunia penjualan manga online, serta terus belajar dan mengembangkan keterampilan saya di dunia perwibuan.

Saya sendiri orangnya sangat periang gembira yang ramah nan baik dan rajin menabung. Saya berharap semua pembeli maupun pembaca di Brandon Manga sudah menebak hal ini sejak mereka baru membuka web ini. Meskipun saya seorang wibu yang tidak terlalu brutal, saya hanya menonton anime sesekali, namun ilmu saya soal anime jangan diragukan lagi. Saya sudah Top Class Graduate dalam dunia perwibuan maupun peranimean
